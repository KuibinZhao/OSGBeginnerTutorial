# OpenSceneGraph 3.0 - Beginner's Guide 源码
* 03_MonitorRFC
	- 查看引用计数的变化
* 03_ReadFromCommandLine
	- 从命令行读取文件
* 03_RedirectNotifier
	- 设置输出调试信息到文件
* 04_ColoredQuad
	- 绘制一个四边形
* 04_DrawOctahedron
	- 绘制一个多面体, 主要是讲图元
* 04_OpenGLTeapot
	- 自定义 drawable, 进行自定义绘制
* 04_PrimitiveFunctor
	- 仿函数, 可以统计多边形面信息
* 04_SimpleObject
	- 几个简单的形状 box, sphere, cone
* 04_TessellatePolygon
	- 凹多边形转凸多边形
* 05_AddModel
	- osg::Group::addChild
